# 公众号 HTML 智能直通

这是一个本地 Chrome / Edge 扩展。它会自动从 HTML 识别标题、作者，并直接写入微信公众号图文编辑器的富文本正文。

## 导入策略

- 不再提供“导入模式”。固定采用智能直通：普通段落、标题、引用、列表、表格仍保留为 HTML 文本，便于继续编辑。
- 已有的 SVG 图层、路径、遮罩和 SMIL 动画会被保留；SVG 内部 ID 会自动隔离，降低多个组件互相冲突的概率。
- CSS 会转换成内联样式，并尽量保留布局、颜色、边框、阴影、定位等信息。
- 正文优先直接恢复已转换好的 DOM，再通知公众号编辑器已修改；这能避开粘贴式写入对内联样式和 SVG 的提前清理。插件会回报实际写入的 SVG 与样式数量。
- 标题和作者只会分别写入可识别的标题栏、作者栏，正文 HTML 不会写入这些字段。
- 当原稿没有指定字体时，正文会补用适合中文阅读的圆润系统字体栈，并开启抗锯齿；原稿已指定的字体、字距不会被覆盖。
- 对明确标记为 `icon`、`glyph`、`symbol`、`data-icon` 或“图标”的小型 SVG、`i`、`span`，会在它本身没有边框或背景时补上柔和的圆角框。普通插画、轮播 SVG 和已有设计的图标不会被套框。

## 动态组件兼容清单

插件会直接保留：

- SVG / SMIL 的轮播、点击切换、弹窗、答题、路径与序列动画。
- GIF 动图、带有效链接的图片与文字、横向或纵向滚动容器。
- 已关联小程序的文字或图片链接（`data-miniprogram-appid` / `data-miniprogram-path`）。
- 从公众号后台或已发布文章复制出的原生组件代码：`mpvoice` 音频、`mpvideo` / 合规视频 iframe、`mp-miniprogram` 小程序、`mp-common-profile` 公众号名片，以及文章、直播、小店商品、红包封面等组件。

下面这些网页能力不会在公众号图文中稳定运行，插件会移除并在转换报告提示：JavaScript、任意 iframe、原生 `video` / `audio`、表单、按钮、Canvas，以及 CSS 关键帧动画。音频、视频、视频号、公众号名片和小程序卡片必须先在公众号后台创建或从已发布文章复制到 HTML；它们依赖公众号账号关联、素材或资源 ID，不能由普通网页代码伪造。

## 必须知道的限制

微信公众号最终仍会进行一次自身的内容清理。任何浏览器扩展都无法保证微信不删除其明确禁止的标签或样式；JavaScript、iframe、表单、视频/音频不会导入。对于已有 SVG 组件，请始终以公众号的手机预览结果为准。

外链图片可能因防盗链失效；Base64 图片也可能被后台清理。需要稳定展示的图片，建议在公众号后台重新上传。

## 安装与使用

1. 打开 Chrome 或 Edge 的扩展管理页，开启“开发者模式”。
2. 点击“加载已解压的扩展程序”，选择本文件夹 `wechat-html-importer-extension`。
3. 登录 `mp.weixin.qq.com`，进入“新建图文”。
4. 点击正文空白区域，使富文本编辑器进入可编辑状态。
5. 打开扩展，选择 HTML 文件或粘贴内容。标题和作者会自动填入，可在导入前手动修正。
6. 点击“导入到当前图文”，再用手机预览检查并手动保存草稿。

若扩展刚更新，请在扩展管理页点击“重新加载”。如果提示未找到正文编辑区，请先进入“新建图文”，点击一次正文空白区域后重试。

## 安全边界

- 所有转换在浏览器本地完成。
- 扩展不要求 AppID、AppSecret，不读取密码，不保存或发布草稿。
- 它只在 `mp.weixin.qq.com` 页面运行，并且只有点击“直通导入”时才修改当前图文页面。
